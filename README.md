## 产品特性

- 任务支持开始、暂停和结束，不只是记录“完成/未完成”。
- 可以记录不同工作类型的实际用时，保留任务执行过程。
- 工作统计支持对比实际用时和预计用时。
- 支持按时间段查看工作类型占比和关键词点云。
- 内置 Agent 能辅助创建任务、读取文档生成任务信息，并拆分复杂任务。
- 支持通过 Agent 基于 TODO 和计时记录生成日报、周报，并复制 Markdown 报告。
