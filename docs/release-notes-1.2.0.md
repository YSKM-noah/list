# it knows you better 1.2.0

本版本新增个人知识库基础功能，让 TODO、Agent 和长期资料沉淀开始形成三模块结构。

## 新增

- 新增 `知识库` 侧边栏页面。
- 支持创建和编辑 Markdown 知识条目。
- 支持标题、正文、项目、标签和来源类型。
- 支持知识条目搜索。
- 支持按标签筛选知识条目。
- 新增本地 `knowledge.json` 持久化文件。

## 说明

- 当前版本先提供基础 Markdown 条目和搜索能力。
- 文档导入、任务关联、Agent 总结和语义检索会放到后续版本。
- 公开下载仓库只包含下载页、版本号和版本说明，不包含应用源码。
- GitHub Release 页面自动显示的 `Source code (zip)` 和 `Source code (tar.gz)` 是本公开下载页仓库的自动快照，不是软件源码。
