# it knows you better 1.0.1

日期：2026-05-31

## 新版本变化

- 修复添加或删除自定义工作类型时窗口/界面重新缩放的问题。
- 普通设置保存不再重新应用窗口尺寸和位置。
- 只有贴片模式、透明度、锁定位置等窗口设置才会重新应用窗口配置。
- 设置页、Agent 页、统计页预留滚动条空间，减少滚动条出现时的布局偏移。

## 产品特性

- 任务支持开始、暂停和结束，可以记录任务执行过程中的实际用时。
- 不同于只记录“完成/未完成”的普通 TODO，it knows you better 会保留每类工作的时间投入。
- 工作统计可以对比实际用时和预计用时，查看工作类型占比和关键词点云。
- Agent 能辅助创建任务、读取文档生成任务信息，并把复杂任务拆分成子任务。
