# it knows you better 1.3.2

本版本把 TODO 从“DDL 查看器”进一步推进到“schedule 助手”，并补充知识库问答能力。

## 新增

- 新增知识库 Agent 问答，可基于本地知识条目回答问题、展示引用和后续问题。
- 新增 `todo.plan_schedule` Agent 技能，用于作业、报告、课程项目等有最终截止时间的任务排期。
- LLM 创建任务新增“作业排期”模式，支持多轮对话澄清作业内容、最终截止时间、可用时间和已完成部分。
- 作业排期结果支持预览后再保存，避免 Agent 结果直接写入 TODO。
- 新增真实截止时间字段 `deadlineTime`，用于和 `deadlineDate` 一起记录最终 DDL。
- 新增应用内 schedule 提醒：计划开始、错过计划块、预计结束检查、计时超时和 deadline 风险。
- 提醒弹窗支持开始工作、确认完成、稍后提醒、继续计时、暂停并重排，以及把未完成任务重排到真实截止前。

## 改进

- `date` / `dueTime` 现在明确表示建议开始执行的时间，不再等同于 DDL。
- `deadlineDate` / `deadlineTime` 表示最终硬约束，只用于风险判断和重排边界。
- 错过计划工作块时会提高任务优先级，并尽量在真实 deadline 前重新安排剩余工作。
- 排期 Agent 会参考已有未来任务，避免把任务全部堆到截止当天。
