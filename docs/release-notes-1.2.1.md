# it knows you better 1.2.1

本版本继续完善知识库，让知识条目可以直接从文档进入系统，并和 TODO 任务建立关联。

## 新增

- 知识库支持导入 PDF、DOCX、TXT 和 Markdown 文档。
- 导入文档会自动填入知识标题、正文、来源类型和 `文档` 标签。
- 知识条目可以搜索并关联已有任务。
- 已关联任务会显示为可移除的任务标签。
- 知识预览中可以查看关联任务，并点击打开任务详情。

## 修复

- 移除了 1.2.0 中重复渲染的知识库页面片段，避免页面结构膨胀和布局不稳定。
