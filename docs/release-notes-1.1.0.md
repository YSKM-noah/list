# it knows you better 1.1.0

这是 v1 之后的第一个 Agent 能力扩展版本，重点是加入日报、周报生成。

## 新增

- 新增工作报告页面。
- 支持选择日报或周报、时间范围和报告风格。
- Agent 会读取 TODO、任务状态、工作类型、标签、预计耗时、实际计时、完成说明和活动记录来生成报告。
- 支持补充额外上下文后生成报告。
- 支持 Markdown 预览和复制。

## 产品特性

- TODO 不只是完成清单，还能记录开始、暂停、结束等执行过程。
- 实际用时会按工作类型统计，后续可以被 Agent 用来生成日报、周报和复盘。
- Agent 与 TODO 保持相对独立，通过 skills 调用 TODO 数据，方便后续继续扩展。

## 说明

- 报告生成需要先在 Agent 配置中启用并配置模型。
- 当前版本暂不保存报告历史，后续会继续补充报告归档、导出和自动生成。
