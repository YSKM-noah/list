# it knows you better 1.2.4

本版本补齐知识库的日常使用体验，重点降低误操作风险，并加强任务与知识之间的工作流。

## 新增

- 知识库草稿新增未保存修改检测。
- 新建、切换和清空知识条目前会提示是否丢弃未保存内容。
- 知识库编辑区显示保存状态。
- 知识列表新增正文摘要，浏览时更容易判断内容。
- 任务详情新增 `新建关联知识`，可直接生成绑定当前任务的知识草稿。
- 任务详情关联知识区新增空状态。
