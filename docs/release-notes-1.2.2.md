# it knows you better 1.2.2

本版本先做前端视觉优化，保持原有功能和布局结构不变，重点提升界面质感和交互反馈。

## 优化

- 统一面板层级、边框、阴影和半透明表面。
- 优化主按钮、次按钮、危险按钮和图标按钮的悬停与焦点状态。
- 优化任务卡片，新增左侧状态强调、卡片悬停反馈和更清晰的层级。
- 优化日历单元格、统计卡片、知识库列表项和设置选项的交互反馈。
- 增加统一输入框焦点状态和滚动条样式。
