# it knows you better 1.1.1

本版本重点处理没有明确截止日期的长期事项，例如学习一门语言、研究一个课题、持续推进某个能力建设。

## 新增

- 新增截止策略：无明确截止、尽快推进、有截止日期。
- 新增任务形态：普通任务、长期目标、持续推进。
- 长期目标支持每周投入时间和复盘间隔。
- 工作统计新增 `近7天` 时间范围。

## 调整

- 新建和编辑任务中，原有“日期”调整为“安排日期”。
- 日报只显示单日选择和今日/选中日快捷按钮。
- 周报只显示起止日期和本周/上周快捷按钮。
- Agent 创建任务、拆分任务、生成报告时会识别长期目标和软截止，不再强行编造硬截止日期。
