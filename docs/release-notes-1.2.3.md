# it knows you better 1.2.3

本版本继续完善知识库，让知识条目更适合长期阅读、筛选和与任务互相跳转。

## 新增

- 知识库新增来源筛选，可按手写、文档、网页摘录、Agent 总结查看。
- 知识库新增排序，可按最近更新、最近创建、标题排序。
- 知识预览从纯文本升级为 Markdown 渲染视图，支持标题、列表、引用、代码块、行内代码和粗体。
- 知识预览新增复制 Markdown 按钮。
- 知识编辑区显示创建时间和更新时间。
- 任务详情中新增关联知识入口，可从任务反向打开相关知识条目。
